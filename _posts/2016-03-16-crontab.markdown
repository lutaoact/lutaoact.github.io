## Linux Crontab Format
{% highlight sh %}
MIN HOUR DOM MON DOW CMD
{% endhighlight %}

### 各字段值允许的范围
{% highlight sh %}
MIN    0-59
HOUR   0-23
DOM    1-31
MON    1-12
DOW    0-6
CMD    command
#DOW: 0表示周日 6表示周六
{% endhighlight %}
