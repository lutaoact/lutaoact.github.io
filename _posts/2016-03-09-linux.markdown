## watch

### 以指定的时间间隔自动地运行指定的命令，默认的时间间隔为 2 秒。
{% highlight sh %}
watch sar -P ALL 1 1 #查看所有cpu的使用情况
watch ls -l #查看文件大小的变化
{% endhighlight %}

## openssl

### 生成文件或字符串的校验和
{% highlight sh %}
openssl sha1/md5 filename #生成指定文件的校验和
echo -n 'xxx' | openssl md5 #生成字符串的校验和
{% endhighlight %}
